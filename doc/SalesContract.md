# Sales Contract

After a package as sold through sales order. it's time to bundle every details in the package into tangible document so both parties can keep one as reference of services and prices that were traded to get the service

sale contract is an A4-format PDF with all of these data appears as page or pages (if data exceed one page)
1. cover page, beautiful cover with text in summary of which who bought what package for whom. and when to start
2. short thank you letter and assurance that customer will get the most satisfied service
3. layout all services user purchased in one table
4. rules of the service and policy, you can study from example in /doc
5. attached receipt from invoice data
